# PROJECT STATUS - Medicube

## Что сделано

- Mini App работает в Telegram через GitHub Pages.
- Реализован раздел `Сегодня` для ухода (утро/вечер) с явным сохранением по кнопкам.
- Добавлен отдельный раздел `Добавки`:
  - `Утро`: Омега-3, Витамин D, B12, B9, Коллаген, Псиллиум, Железо
  - `День`: Омега-3, Витамин D, B12, B9, Коллаген, Псиллиум, Железо
  - `Вечер`: Магний
  - отдельное сохранение: `Сохранить утро`, `Сохранить день`, `Сохранить вечер`
- История расширена:
  - блок `Уход` (утро/вечер)
  - блок `Добавки` (утро/день/вечер)
- Календарь учитывает активность и по уходу, и по добавкам.
- Backend и БД расширены:
  - новые модели Prisma: `Supplement`, `SupplementEntry`
  - новые API: `GET /api/supplements`, `GET/PUT /api/supplements/day`
  - `GET /api/calendar/month` и `GET /api/calendar/history` учитывают добавки
- Сегрегация данных по пользователям сохраняется:
  - фильтрация по `telegram user id` из проверенного `initData`
  - пользователи не видят чужие записи.
